# CLAUDE.md — Vans Parts Midlands

## Ce este acest proiect
Website static (HTML + CSS + JavaScript) pentru **Vans Parts Midlands**, un
service / furnizor de piese de duba din UK. Nu există framework, build sau
backend — sunt fișiere `.html` care se deschid direct în browser.

**Tip proiect:** client (website agenție To The Sky).

## Structură
- `index.html` și restul paginilor `.html` — paginile site-ului (home, despre,
  piese, reparații, contact, testimoniale, termeni, cookies, confidențialitate).
- `css/style.css` — toate stilurile.
- `js/` — scripturi: `main.js`, `cookie-banner.js`, `testimonials-carousel.js`.
- `images/` și fișierele `.png` / `.jpg` din rădăcină — imagini (loguri mărci,
  duburi, poze workshop).
- `uploads/` — imagini sursă / variante de lucru.
- `brand-spec.md` — specificația de brand (culori, fonturi). Sursa de adevăr
  pentru design: negru `#0A0A0A`, roșu accent `#CC1010`, font display `Oswald`.

## Cum lucrezi aici
- Respectă culorile și fonturile din `brand-spec.md` la orice modificare vizuală.
- E un site static: pentru a-l vedea, deschizi `index.html` în browser.
- Textele sunt în engleză (audiență UK). Păstrează limba și tonul existent.
